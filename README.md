# Printf

## Description

The ft_printf project consists of recreating the behavior of the standard C function `printf`.

The goal is to understand and implement:
- variable argument handling using `stdarg.h`
- format string parsing
- output of different data types

---

## Prototype

```c
int ft_printf(const char *format, ...);
```
